# 박스비트 (BoxBeat)

크로스핏 동작으로 이루어진 링피트 어드벤처 스타일 리듬게임 정적 웹앱입니다.

## 포함 기능
- 관절 기반 옆모습 SVG 캐릭터가 실제 동작 자세(짐네스틱 4종 + 역도 12종, 총 16개 동작)를 박자에 맞춰 수행
- 월드맵 진행 구조 — 짐네스틱 월드 / 역도 월드, 스테이지를 클리어해야 다음 칸이 열림
- 적 배틀 연출 — PERFECT/GOOD 판정마다 적 캐릭터에게 데미지
- 개별 동작 스테이지 + WOD 조합 보스전(프란/그레이스/머프/DT 스타일)
- Web Audio API로 실시간 합성한 비트/타격음 (외부 음원 파일 불필요)
- PERFECT / GOOD / MISS 판정, 콤보, HP, 정확도 기반 랭크(S~F)
- 스테이지별 최고 점수, 클리어 진행도, 누적 XP/레벨 localStorage 저장

## 바로 실행
- `index.html` 을 브라우저에서 열면 바로 플레이할 수 있습니다.

## 플레이 방법
1. 모험 지도에서 열려 있는 스테이지를 선택합니다.
2. 캐릭터가 박자에 맞춰 동작을 반복하고, 가장 낮은 자세(바닥)에 닿는 순간 화면을 탭하거나 스페이스바를 누릅니다.
3. 타이밍이 정확할수록 PERFECT, 다소 늦거나 빠르면 GOOD로 판정되며 적에게 데미지를 줍니다. 놓치면 MISS로 내 HP가 줄어듭니다.
4. HP가 0이 되면 실패, 스테이지를 클리어하면 지도의 다음 칸이 열립니다.

# 박스비트 (BoxBeat)

크로스핏 동작(스쿼트·버피·박스점프·스내치)으로 이루어진 리듬게임 정적 웹앱입니다.

## 포함 기능
- 4레인 노트 리듬게임 (키보드 D·F·J·K 또는 화면 터치 버튼)
- 난이도별 WOD 트랙 4종 (BPM 100~168)
- Web Audio API로 실시간 합성한 비트/타격음 (외부 음원 파일 불필요)
- PERFECT / GOOD / MISS 판정, 콤보, HP, 정확도 기반 랭크(S~F)
- 트랙별 최고 점수 및 누적 XP/레벨 localStorage 저장

## 바로 실행
- `index.html` 을 브라우저에서 열면 바로 플레이할 수 있습니다.

## 플레이 방법
1. 트랙을 선택하면 박자에 맞춰 동작 노트가 판정선까지 내려옵니다.
2. 노트가 판정선에 닿는 순간 같은 레인의 키(D/F/J/K) 또는 하단 버튼을 누릅니다.
3. 타이밍이 정확할수록 PERFECT, 다소 늦거나 빠르면 GOOD, 놓치면 MISS 처리됩니다.
4. HP가 0이 되면 실패로 곡이 종료됩니다.
